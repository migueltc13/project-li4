Edits the auction if all requirements are met. 

Checks if the auction exists and belongs to the client. If so, updates the auction details as long as they meet the validation requirements\+: 
\begin{DoxyItemize}
\item Minimum bid must be greater than 0. 
\item Title must be between 1 and 64 characters. 
\item Description must be between 1 and 2048 characters. 
\item Images must be valid. 
\end{DoxyItemize}If the auction details are updated successfully, redirects to the edit auction page with a success message. Otherwise, redisplays the edit auction page with an error message. \begin{DoxyReturn}{Returns}
A task that represents the action of editing the auction.
\end{DoxyReturn}
