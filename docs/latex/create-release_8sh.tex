Shell script to publish a .NET application. 

This script automates the process of publishing a .NET application using the \textquotesingle{}dotnet publish\textquotesingle{} command, it also ensures that the application is built in Release mode.\hypertarget{create-release_8sh_usage}{}\doxysubsection{Usage}\label{create-release_8sh_usage}

\begin{DoxyItemize}
\item Ensure that the \textquotesingle{}dotnet\textquotesingle{} CLI is installed on your system.
\item Run the script in the terminal at the {\ttfamily Better\+Finds} root directory.
\item The script will build and publish the application in Release mode.
