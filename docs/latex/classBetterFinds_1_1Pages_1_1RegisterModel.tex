Creates a new account if all requirements are met. 


\begin{DoxyItemize}
\item Username must be unique, and between 3 and 32 alphanumeric characters. 
\item Password must be between 8 and 64 characters. 
\item Password must match Confirm\+Password. 
\item Email must be unique, and between 5 and 320 characters. 
\item Full name must be between 3 and 64 alphabetical characters and spaces. 
\item Profile picture must be 256 characters or fewer. 
\item Opt\+Newsletter is an optional checkbox. 
\end{DoxyItemize}If the requirements are met, the user is added to the database and redirected to the login page, with a success message and the username pre-\/filled. Otherwise, the user is redirected to the register page with an error message.
