Creates a new auction if all requirements are met. 


\begin{DoxyItemize}
\item The starting price must be greater than 0. 
\item The minimum bid must be greater than 0. 
\item The title must be between 1 and 64 characters long. 
\item The description must be between 1 and 2048 characters long. 
\item The end time must be a valid date and time and greater than the current time. 
\item The images must be valid, if any. See \mbox{\hyperlink{classBetterFinds_1_1Utils_1_1Images_aa8f2fa6a396d83165bee82589eadbc38}{Utils.\+Images.\+Is\+Valid\+Images}}. 
\end{DoxyItemize}If all requirements are met, the auction is created and the user is redirected to the new auction page. Otherwise, the user stays on the page and an error message is displayed. 

The users on the \mbox{\hyperlink{classBetterFinds_1_1Pages_1_1IndexModel}{Index\+Model}} or \mbox{\hyperlink{classBetterFinds_1_1Pages_1_1SearchModel}{Search\+Model}} pages are notified that there\textquotesingle{}s updates. 

The auction is added to the background service to check for its ending. 

\begin{DoxyReturn}{Returns}
A task that represents the action of creating a new auction.
\end{DoxyReturn}
