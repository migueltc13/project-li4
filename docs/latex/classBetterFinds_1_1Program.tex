The main entry point of the application. 

It\textquotesingle{}s responsible for configuring services used by the application and starting it. 

Configuration includes\+: 
\begin{DoxyItemize}
\item Razor pages 
\item Authentication 
\item Session 
\item Localization 
\item Auctions background service 
\item Bidders groups service 
\item SignalR 
\item Error handling and page redirection 
\end{DoxyItemize}Additionally, it initializes the Auctions and Bidders groups services, and maps the SignalR hub. 

The Auction background service is started by the \mbox{\hyperlink{classBetterFinds_1_1Services_1_1AuctionBackgroundService}{Services.\+Auction\+Background\+Service}} class and will run asynchronously throughout the application\textquotesingle{}s lifetime to monitor ending auctions.
