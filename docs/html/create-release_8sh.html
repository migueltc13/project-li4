Detailed Description

Shell script to publish a .NET application.

This script automates the process of publishing a .NET application using the 'dotnet publish' command, it also ensures that the application is built in Release mode.

Usage

  • Ensure that the 'dotnet' CLI is installed on your system.
  • Run the script in the terminal at the BetterFinds root directory.
  • The script will build and publish the application in Release mode.

Dependencies

  • dotnet CLI (https://dotnet.microsoft.com/download)

Notes

  • This script assumes the presence of a valid .NET project file.
  • Make sure to customize the script based on your project structure.
